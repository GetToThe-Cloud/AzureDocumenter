Azure Landing Zone Documenter

✓ Available Now

The Landing Zone Documenter is a powerful PowerShell-based tool that automatically inventories your entire Azure Landing Zone infrastructure. It connects to your Azure subscription, scans all resources, and generates a comprehensive HTML report with interactive visualizations.

🎯 Perfect for: Cloud architects, DevOps engineers, and IT auditors who need to document hub-spoke network topologies, maintain compliance records, or onboard new team members quickly with comprehensive infrastructure documentation.

📋 What Gets Documented

  • Network Architecture: Virtual networks, subnets, peering connections, and network topology diagrams
  • VPN Infrastructure: VPN gateways, local network gateways, connections, and IPsec settings
  • DNS Management: Private DNS zones, zone links, and DNS configurations
  • Security Policies: Network Security Groups, Application Security Groups, and firewall rules
  • Resource Organization: Resource groups, tags, naming conventions, and organizational structure
  • Governance: Azure Policy assignments, RBAC roles, and compliance status
  • Monitoring: Log Analytics workspaces, diagnostic settings, and monitoring configurations

🚀 Key Features

  • Auto-discovery of all Landing Zone components
  • Interactive HTML reports with search and filtering
  • Network topology visualization
  • Export to PDF
  • Multi-subscription support
  • Compliance and security audit reports

🔧 Technology Stack

PowerShell 7+ Azure PowerShell HTML5/CSS3 REST API

AVD Environment Documenter

✓ Available Now

The AVD Environment Documenter provides complete documentation of your Azure Virtual Desktop deployment. It analyzes host pools, session hosts, application groups, and user assignments to create detailed reports with health status, capacity planning insights, and optimization recommendations.

🎯 Perfect for: AVD administrators, support teams, and MSPs who need to track session host health, monitor capacity, document user assignments, or troubleshoot AVD issues with comprehensive environment insights.

📋 What Gets Documented

  • Host Pools: Configuration details, load balancing settings, and capacity information
  • Session Hosts: VM details, status, agent versions, and performance metrics
  • Application Groups: Desktop and RemoteApp configurations with user assignments
  • Workspaces: Workspace settings and application group associations
  • User Sessions: Active sessions, user profiles, and connection history
  • FSLogix Profiles: Profile storage configurations and capacity usage
  • Networking: Virtual network integration and connectivity details
  • Installed Applications: Software inventory across all session hosts

🚀 Key Features

  • Real-time health status monitoring
  • Capacity planning and utilization reports
  • User assignment and licensing overview
  • Session host comparison matrix
  • Performance baseline recommendations
  • Application installation automation scripts

🔧 Technology Stack

PowerShell 7+ Az.DesktopVirtualization HTML5/JavaScript Azure REST API
